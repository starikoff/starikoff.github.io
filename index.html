Let Java Help You

Introduction

In this article I will briefly discuss several techniques and practices that I adopted over the years and find useful. They are all targeted at using the powers of Java -- the language, the platform, the ecosystem — to create more correct software systems, easier and faster.

These techniques are applicable to the basic Java and not limited to "Java for the Web", "Java with Spring" or anything.
